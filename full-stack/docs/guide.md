# Dawin Dash 使用指南

本指南適用於行銷團隊成員，說明如何操作 Dawin Dash 專案管理系統。

---

## 產品功能介紹

### 這個系統是什麼？

Dawin Dash 是一套專為影視 IP 專案打造的管理工具。你可以在上面建立多個專案，追蹤每個專案底下的工作項目、指派負責人、設定時程，並透過甘特圖一目了然地掌握整體進度。

### 功能總覽

- **多專案管理** — 同時管理多個影視 IP 專案，自由切換
- **任務追蹤** — 建立任務、設定狀態（待辦 / 進行中 / 已完成等）、優先級（高 / 中 / 低）
- **子任務** — 將大任務拆分為小步驟，逐項勾選完成
- **甘特圖時間軸** — 以時間軸方式查看所有任務的排程
- **檔案管理** — 在任務底下上傳、下載相關檔案
- **CSV 匯入匯出** — 批次匯入任務或匯出報表
- **負責人指派** — 為每個任務或子任務指定負責人
- **分類標籤** — 將任務歸類（商務合作、活動、行銷等）
- **深色 / 淺色模式** — 可依個人喜好切換介面風格
- **拖曳排序** — 直接拖動子任務調整順序

### 支援的瀏覽器與裝置

- **電腦**：Chrome、Edge、Safari、Firefox（建議使用最新版本）
- **手機 / 平板**：支援行動裝置瀏覽，甘特圖會自動切換為適合觸控的列表模式

## 操作教學

### 如何登入系統

1. 在瀏覽器輸入系統網址
2. 輸入你的 Email 和密碼
3. 點擊「登入」按鈕
4. 首次登入會被要求設定新密碼，輸入新密碼後即可進入系統

### 如何查看 / 切換專案

1. 登入後，左側欄會顯示所有專案列表
2. 點擊任一專案名稱即可切換到該專案
3. 點擊左上方的選單按鈕可以收合 / 展開側邊欄

### 如何排序專案卡片

1. 在左側欄的專案列表中，你可以透過兩種方式調整專案順序：
   - **拖曳排序**：長按專案卡片，拖動到想要的位置放開即可
   - **條件排序**：點擊排序按鈕，選擇依「名稱」、「日期」或「進度」自動排序
2. 排序結果會自動儲存，下次登入時維持你設定的順序

### 如何建立新任務

1. 切換到目標專案
2. 在任務列表上方找到「新增任務」按鈕
3. 在跳出的視窗中填寫：
   - **任務名稱**（必填）
   - **狀態** — 待辦、進行中、已完成、提案中、待確認
   - **優先級** — 高、中、低
   - **分類** — 商務合作、活動、播出/開始、行銷、發行、市場展
   - **開始日期** 和 **結束日期**
   - **負責人**
   - **備註**
4. 點擊確認按鈕儲存

### 如何編輯任務

- **直接在表格中編輯**：點擊任務的任何欄位（名稱、狀態、優先級、日期、負責人等），直接修改，修改後自動儲存
- **狀態切換**：點擊狀態欄位，從下拉選單中選擇新狀態
- **優先級調整**：點擊優先級欄位，選擇高 / 中 / 低
- **日期設定**：點擊日期欄位，從日曆中選擇日期

### 如何管理子任務

1. 展開一個任務（點擊任務列左側的展開箭頭）
2. 在子任務區域點擊「新增子任務」
3. 輸入子任務名稱和負責人
4. **勾選完成**：點擊子任務前方的核取方塊，即標記為完成
5. 子任務的完成比例會自動計算並顯示在進度條上
6. 可以拖曳子任務來調整排列順序

### 如何指派負責人

每個任務和子任務都可以指定負責人，負責人名單來自系統中的使用者帳號：

1. **單一任務指派**：在任務的「負責人」欄位點擊，會出現下拉選單，列出所有系統使用者，選擇即可
2. **子任務指派**：展開任務後，在每個子任務的負責人欄位同樣有下拉選單可供選擇
3. **批次指派**：在 Data 分頁中，勾選多個任務後，可以一次將選中的任務全部指派給同一位負責人
4. 當管理員在使用者管理頁面修改某位使用者的姓名時，該使用者作為負責人的顯示名稱會自動更新

### 如何上傳 / 下載檔案

1. 展開任務，找到檔案管理區域
2. **上傳**：點擊「上傳檔案」按鈕，選擇要上傳的檔案
3. **下載**：在已上傳的檔案列表中，點擊檔案名稱即可下載
4. **刪除**：點擊檔案旁的刪除按鈕移除檔案

### 如何查看甘特圖時間軸

1. 在主畫面上方的分頁切換中，選擇「Timeline」分頁
2. 甘特圖會顯示所有任務的時間排程
3. 橫軸為時間、縱軸為各任務
4. 不同顏色代表不同專案
5. 可以左右捲動查看不同時間區段

### 如何匯出 CSV

1. 在主畫面找到 CSV 匯出按鈕
2. 點擊後，系統會將目前的任務資料匯出為 CSV 檔案
3. 下載的檔案可用 Excel 或 Google Sheets 開啟

### 如何設定專案圖示

每個專案可以設定一個代表圖示，方便快速辨識：

1. 在左側欄的專案卡片上，找到專案名稱左方的圖示區域（預設會顯示專案名稱的第一個字）
2. **上傳圖示**：點擊圖示區域，從電腦選擇一張圖片上傳（支援 JPG、PNG、WebP、GIF，大小限制 5 MB）
3. **刪除圖示**：將滑鼠移到圖示上方，會出現一個「×」刪除按鈕，點擊即可移除
4. 圖示會自動儲存到雲端，下次登入時依然會顯示

### 如何管理系統備份（超級管理員）

超級管理員可以設定自動備份，保護系統資料：

1. 在左側欄底部找到「備份管理」入口，點擊進入備份管理頁面
2. **設定備份目標**：
   - **Cloudflare R2**：輸入 Account ID、Bucket、Access Key、Secret Key，點擊「測試連線」確認設定正確
   - **Google Drive**：輸入 Service Account Email、Private Key、Folder ID，點擊「測試連線」確認
3. **設定自動排程**：
   - 開啟自動備份開關
   - 選擇備份頻率（每日 1 次至 4 次）
   - 勾選備份目標（R2 和 / 或 Google Drive）
   - 設定保留份數（預設 30 份，超出會自動刪除最舊的）
4. **手動備份**：隨時點擊「立即備份」按鈕執行一次備份
5. **查看備份歷史**：表格中會顯示每次備份的時間、目標、檔案大小、耗時和狀態
6. **查看操作記錄**：切換到「操作記錄」分頁，可以看到所有敏感操作的紀錄（誰在什麼時候做了什麼）

### 如何切換深色 / 淺色模式

1. 在畫面中找到主題切換按鈕
2. 點擊即可在「明亮模式」和「柔和深色模式」之間切換
3. 你的選擇會被記住，下次登入時自動套用

## 常見問題 FAQ

### 忘記密碼怎麼辦？
請聯繫系統管理員（super_admin 帳號持有人），由管理員重設你的密碼。

### 檔案上傳有大小限制嗎？
有，單個檔案大小限制視伺服器設定而定。如果上傳失敗，請嘗試壓縮檔案後再上傳，或聯繫管理員。

### 可以同時多人使用嗎？
可以。多位團隊成員可以同時登入並操作系統。但請注意，如果兩人同時編輯同一個任務，後儲存的內容會覆蓋先前的修改。

### 資料會自動儲存嗎？
是的。當你在表格中修改任何欄位（狀態、優先級、日期等），修改會立即自動儲存。如果儲存失敗，畫面底部會出現紅色提示訊息。

### 手機可以用嗎？
可以。系統支援手機瀏覽器存取。甘特圖在手機上會自動切換為列表模式，方便觸控操作。建議複雜的編輯操作在電腦上進行。

### 不同角色有什麼差異？
- **管理員**：可以管理自己負責的專案和任務
- **超級管理員**：除了一般功能外，還可以管理使用者帳號、清除所有任務等進階操作
